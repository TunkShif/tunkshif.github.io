Anki Azul Template
================

[Anki][1] 是一款开源的多平台的用来记忆的单词卡软件, 其卡片模板可以用 `html css js` 来实现丰富的自定义功能.

[Anki Azul Template][2] 是个人所用的一套风格统一的模板.
