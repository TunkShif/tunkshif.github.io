Linux Tricks

Merge PDF

Requirement: ghostscript

gs -dNOPAUSE -sDEVICE=pdfwrite -sOUTPUTFILE=output.pdf -dBATCH first.pdf second.pdf

Parse JSON

Requirement: jq

curl -s https://v1.hitokoto.cn/ | jq '.hitokoto'

Clipboard

Requirement: xclip

pwd | xclip -selection 'clipboard' # copy current path
